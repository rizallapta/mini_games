Dasar-dasar Perintah Git

Berikut adalah perintah dasar Git yang penting untuk pemula:

  • git init: Membuat repository Git baru.
  • git clone: Mengunduh repository dari remote server.
  • git add: Menambahkan file ke staging area.
  • git commit: Menyimpan perubahan ke dalam repository dengan pesan.
  • git status: Menampilkan status file di repository.
  • git push: Mengunggah commit dari repository lokal ke remote.
  • git pull: Mengambil dan menggabungkan perubahan dari remote ke lokal.
  • git branch: Membuat atau menampilkan cabang dalam repository.
  • git checkout: Beralih ke cabang atau commit tertentu.
  • git merge: Menggabungkan dua cabang.
